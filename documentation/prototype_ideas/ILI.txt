preformater --> analysis --> organizer --> executor

preformater: will convert any text into the standardized format for the
analyzer or for the analysis phase.

analysis/er: have two parts static analysis and syntax analysis \
  syntax analysis will check if the command's and order of the command is
  valid.
  static analysis will check if the logic and the variable are valid.

organizer: will order all instructions in stack machine instructions set or in
dequeue format.

executor: will execute the instructions in the stack machine instructions set.
