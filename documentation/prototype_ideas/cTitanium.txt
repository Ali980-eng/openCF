the compiler will provide a overloading by using the name manglaing.
the key word prefix will be like to the namespace in c++.
every function has not a return type the type will be void by defalut.
every thing in language is object unless spcified other wise by ().

postfixes for name manglaing:
b = bool
c = char
s = short
i = int
l = long
f = float
d = double
ll = long long
li = long int
ld = long double

example:
    short add(short a, short b) {
        return a + b;
    }

    compiler equ =

        short add_ss(short a, short b) {
            return a + b;
        }

    int add(int a, int b) {
        return a + b;
    }

    compiler equ =

        int add_ii(int a, int b) {
            return a + b;
        }
