Test Stream Structure

The code defines a test_stream structure, which contains information about each test result. It has three fields:

details: a boolean indicating whether detailed testing results are desired
seplen: an unsigned integer representing the expected number of spaces between characters (used for formatting purposes)
sepch: a character representing the separator to use when concatenating strings

Adding Test Results

The code provides several functions that allow you to add test results to a test_stream structure:

addTestBool: adds boolean test results
addTestChar: adds character test results
addTestShort, addTestInt, addTestLong, and addTestFloat: adds specific types of integer values (short, int, long, double) to the test stream
addTestSize: adds size_t value test results

Each function takes a test stream structure as an argument and performs the necessary operations:

For boolean tests: sets success and failed counts based on expected values and checks for valid success/failure counts
For character tests: sets success, failed, and total count counters based on actual characters compared to expected strings
For specific integer types:
	addTestShort, addTestInt: sets success and failed counts based on comparison with expected values
	addTestLong and addTestFloat: sets success and failed counts based on comparison with expected values, using a floating-point arithmetic approach to avoid truncation

Calculating Test Statistics

The code provides several functions that calculate various statistics from the accumulated test results:

ASR: calculates the average success rate (0.0-100.0) as a percentage
AFR: calculates the average failure rate (0.0-100.0) as a percentage

Printing Test Summary Report

The code defines a function print that prints a comprehensive test summary report, including statistics for success and failed tests, total count, ASR, and AFR.
