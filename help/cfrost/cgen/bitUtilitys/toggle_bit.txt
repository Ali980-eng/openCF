clite/bitUtilitys/toggle_bit.h
    This header file provides functions to toggle (flip) bits in various integer types.
    Functions:
        Toggle Bit at Specified Position in Various Integer Types
        toggle_u8bit: Toggles a bit at position 0-7 of an unsigned 8-bit integer.
        toggle_u16bit, toggle_u32bit, toggle_u64bit: Toggles bits at positions 0-15, 0-31, and 0-63 respectively of an unsigned 16-bit, 32-bit, or 64-bit integer.
        toggle_8bit, toggle_16bit, toggle_32bit, toggle_64bit: Toggles a bit at position 0-7 (unsigned), 0-15 (signed), 0-31 (signed), and 0-63 (signed) of an unsigned or signed integer respectively.
        toggle_char_bit and toggle_short_bit: Toggles bits at positions 0-7 and 0-15 (for short integers) of a character to 1.
        toggle_int_bit and toggle_long_bit: Toggles bits at positions 0-31 (unsigned), 0-63 (signed), and 0-63 (signed) respectively of an integer to 1.
    Exceptions:
        If the specified position is out of range, an "OutOfRange Error" message is printed and the function returns the original value.
        Toggling always flips the bit; no additional error condition is raised for the current bit state.
