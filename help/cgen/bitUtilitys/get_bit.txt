clite/bitUtilitys/get_bit.h
    Overview
        This header provides a comprehensive set of functions for reading or retrieving the value of individual bits at specified positions from various integer types.
        It supports both signed and unsigned integer types ranging from 8-bit to 64-bit, as well as convenience wrappers for common types like char, short, int, and long.
    Functions
        get_u8bit
            Get bit value at specified position from 8-bit number
            Parameters:
                num: The value to read (8-bit integer)
                position: Bit position (0-7)
            Returns: True if the bit is 1, false if it's 0; out of range error and false if invalid position
            Example:
            uint8_t num = 0b10101010;
            bool result = get_u8bit(num, 7);
        get_8bit
