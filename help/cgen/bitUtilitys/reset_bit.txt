clite/bitUtilitys/reset_bit.h
    Overview
        The reset_bit.h file provides a set of functions to reset (clear to 0) bits at specified positions in various integer types.
        This includes 8-bit, 16-bit, 32-bit, and 64-bit signed and unsigned integers.
    Functions
        uint8_t
            reset_u8bit(uint8_t num, uint8_t position)
                Clears the bit at the specified position in the 8-bit integer.
            Example: uint8_t num = 5; uint8_t position = 1; uint8_t cleared_num = reset_u8bit(num, position);
